Github+Hexo搭建博客

发表于 2017-01-06

准备工作

  • 配置 npm，利用npm直接从网络上获取资源
    • 通过 npm -v查看版本号
  • 下载 git 的安装包，正常安装
    *在文件夹下右击会看到 ‘Git Bash Here’，就能在当前文件夹下启动 git
  • github
    • 注册 github 账号
    • 新建一个仓库，给仓库起一个名字，名字有一定的规范，账号名 + ‘.github.io’

开始搭建

  • 下载 hexo ，(若网络受限，可以通过淘宝的npm服务器下载)
    • 耐心等待，会看到下面的目录结构，直到资源下载完成
    • 下载完成后，通过 hexo -v 查看版本号，若能显示版本号就表示 hexo 下载完成
  • 在 github 上部署 hexo
    • 打开 hexo 的官方网页查看部署的方法hexo.io
    • 本次是使用 git 部署，所以安装 hexo-deployer-git
    • npm 会在你的当前目录下安装 node_modules 文件
  • 初始化博客必须的一些内容
    • 使用 hexo init 命令
    • 等待，随后 hexo 会在当前目录下生成一些文件，文件目录如下图所示
    • 文件目录加载结束后，等待，直到出现如下字样
  • 写博客
    • 使用命令 hexo new 我的博客，新建一个.md文件
    • 对 我的博客.md 进行编辑
  • 查看本地博客
    • 通过命令 hexo server 创建本地静态页面，并打开浏览器到 http://localhost:4000/. 下查看
  • 将博客托管到 github
    • 再次安装 hexo-deployer-git
    • 本次安装和上次安装出现的提示会不太一样。同时，在文件夹下会新增一个 .deploy_git 文件夹
    • 修改配置文件，打开 blog 文件夹下的 _config.yml 文件
    • repo 这个属性的值为你创建后仓库后生成的地址
    • 通过命令 hexo g 生成静态页
    • 通过命令 hexo d 部署到 github
    • 出现如下提示表示部署成功
    • 访问你的仓库名就能进入你的博客
  • 博客主题样式
    • 可以通过下载基于hexo的主题，然后修改配置文件，重新编译你的博客 hexo g -d 就行了
