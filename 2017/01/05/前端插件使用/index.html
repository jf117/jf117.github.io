jquery.cookie

基本作用

  • 通过使用 jqury.cookie.js 小插件对 cookie 进行读、写、删、设置生命周期和 cookie 作用域
  • 该插件返回的 cookie 的类型是对象

基本使用

使用前准备

  • 该插件是基于 jquery 的，所以在使用前应该先引入 jquery.js，然后再引入 jquery-cookie.js

nprogress

下载地址

nprogress.js

基本作用

  • 在网页上方出现一个进度条，用于网页加载时使用
  • 小进度条可用于 Ajax

基本使用

使用前准备

  • 该插件支持 AMD 模块化
  • 在使用前需要引入 nprogress.css 和 nprogress.js

用法

  • 基本用法

    • 简单地调用 start() 和 done() 方法就可以控制进度条

    • 配合 Ajax 可以实现加载 ajax 时出现进度条

  • 配置nprogress

bootstrap-datepicker

下载地址

bootstrap-datepicker下载

文档地址

bootstrap-datepicker文档

基本作用

  • 日期控件，日历选择器

基本使用

使用前准备

  • 从github上下载下来的 bootstrap-datepicker-master 是整个工程文件，我们所需要的文件一般都在 dist 文件夹中
  • 在 css 文件夹中，.map 文件是 less 编译成 css 后编译工具自动生成的对应文件。如果直接引入 .standalone.css 文件，就不需要引入 bootstrap.css 文件；否则就要引入 bootstrap.css 文件
  • 该插件是基于 bootstrap 的，所以在使用前应该先引入 bootstrap.js ，然后再引入 bootstrap-datepicker.js

用法

  • 基本用法

    • 通过 javascript 调用 datepicker

    • 通过标签调用 datepicker

  • 高级定制

    • 通过 javascript

    • 通过标签
