Vue过滤器

发表于 2017-01-31 | 分类于 Vue

过滤器

基本语法: {{ msg | filter }}

内置过滤器

过滤器 作用
capitalize 首字母大写
uppercase 转化为大写字母
lowercase 转化为小写字母
currency 转成货币的方式显示
pluralize 不理解
json 转换成json数据
debounce 延时执行
limitBy 取出字符串或数组的前（正数）几位或后（负数）几位
filterBy 在给定数组中选择满足条件的一个子集，并返回一个新数组，其条件可以是一个字符串、对象、函数
orderBy 对数组进行排序，第二个参数可控制方向

详见Vue过滤器

  • 过滤器可以串联
    • {{ msg | filter1 | filter2 }}
  • 过滤器也接受参数
    • {{ msg | filter1 ‘arg1’ arg2 }}
  • 过滤器函数始终以表达式的值作为第一个参数。带引号的参数视为字符串，不带引号的参数按表达式计算。
