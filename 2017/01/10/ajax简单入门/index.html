ajax简单入门

发表于 2017-01-10 | 分类于 ajax

Ajax编程

简介

  • 全称：Asynchronous Javascript And XML
  • AJAX不是一门的新的语言，而是对现有持术的综合利用，其本质是在HTTP协议的基础上以异步的方式与服务器进行通信

异步

  • 异步的优势在于不阻塞程序的执行，从而提升整体执行效率
  • 独立是异步的前提，耗时是异步的理由

XMLHttpRequest

XMLHttpRequest是浏览器内置对象，用于在后台与服务器通信(交换数据)，由此我们便可实现对网页的部分更新，而不是刷新整个页面

属性

  • readyState:类型short，只读的。

    代表者当前XMLHttpRequest对象的目前的执行状态

  • responseText：类型string，只读。

    存放着服务器返回的响应正文内容（文本内容）

  • responseXML:类型Document，只读

    存放着服务器返回的响应正文内容（XML内容）

  • status：类型short，只读

    服务器响应状态码

  • statusText：类型string，只读

    服务器响应码描述

####方法

  • void abort():取消。回到最出始的状态。
  • String getAllResponseHeaders():获取所有的响应消息头。用逗号分隔的。
  • String getResponseHeader(String headerName):获取指定的响应消息头的值。
  • void open(String method,String url,boolean isAnsy);建立与服务器的链接。
  • method:请求方式 url：请求的地址 isAnsy：是否是异步的。默认是异步的
  • void send(String data):向服务器发送请求正文
  • void setRequestHeader(String headerName,String value):设置请求消息头。

事件

  • onreadystatechange

    当XMLHttpRequest对象的readyState发生变化时，都会调用onreadystatechange指定的方法。

原生ajax编程

  • 定义一个对象var xhr = new XMLHttpRequest();
  • 设置请求行
    • post设置请求行xhr.open('post'，'请求地址'，是否异步[true/false])
    • get设置请求行xhr.open('get'，'请求地址?name=itcast&age=10');
  • 设置请求头xhr.setRequestHeader('Content-Type','application/x-www-form-urlencoded')
    • post提交方式必须写，get则可以不写
  • 设置请求正文
    • post设置请求正文xhr.send('name=itcast&age=10')
    • get设置请求正文xhr.send(null);

  • 监听状态变化的事件

  • 兼容IE5、IE6

$.ajax编程

jQuery为我们提供了更强大的Ajax封装

  • 调用ajax的三种方式：
    • $.ajax({}) 可配置方式发起Ajax请求
    • $.get() 以GET方式发起Ajax请求
    • $.post() 以POST方式发起Ajax请求
    • $.get()和$.post()不用传入一个对象，没有$ajax那么多的方法，只有url、data、success
  • $.ajaxSetup({type:"post"})
    • 给所有的ajax请求设置为post方式，默认为get
    • 该方法可设置ajax的默认属性方法，但是会被ajax、get、post三种方式覆盖
  • $('form').serialize()
    • 获取表单的输入（表单项要有name属性）
    • 序列化表单（即格式化key=val&key=val）
    • 一般会使用return false清除表单提交时的默认刷新页面
  • 属性设置
    • url 接口地址
    • type 请求方式
    • timeout 请求超时
    • dataType 服务器返回格式
    • data 发送请求数据
  • 方法设置
    • beforeSend: function () {} 请求发起前调用
    • success: function (result) {console.log(result)} 成功响应后调用
    • error: function (err) {console.log(err)} 错误响应时调用
    • complete 响应完成时调用（包括成功和失败）

  • 事件

    ajax从发起到结束，要经历若干阶段，根据开发需要，不同阶段可能需要进行不同的业务处理，jQuery就将不同阶段封装成不同的事件

    • ajaxStart:在jq1.8版本之后，绑定事件只能绑定到document上
    • ajaxSend
    • ajaxSuccess
    • ajaxError
    • ajaxComplete:当前ajax请求完成
    • ajaxStop：页面全部ajax请求完成
  • 注意

上述发送方式的数据就是FormData
从表面上看post就是formdata，但不总是这样
content-type：application/json，就是json格式的数据

上述发送方式的数据就是queryString

表单上传文件

  • 普通表单可以上传文件，但是会引起页面的刷新，在实际开发中用户体验不佳
  • 使用XMLHttpRequest对象也可以发起请求，但是低版本浏览器XMLHttpRequest并不支持上传文件，只有高版本浏览器才可以
  • 比较通用的方法是使用flash进行上传，一般都是封装好的插件，例如uploadify和web uploader，并且类似的插件一般都支持html5和flash
