Vue指令和计算属性

发表于 2017-02-01 | 分类于 Vue

指令

v-开头的都是指令

  • v-bind 缩写：就是直接去掉v-bind，保留：和属性
  • v-text | v-html
  • v-if | v-else | v-else-if
  • v-for
  • v-on 缩写：v-on:写成@
  • v-model
  • …

计算属性

  • 为什么Vue.js将绑定表达式限制为一个表达式？

    • 在模板中表达式非常便利，但是它们实际上只用于简单的操作。
    • 模板是为了描述视图的结构。
    • 在模板中放入太多的逻辑会让模板过重且难以维护。

  • 如果需要多于一个表达式的逻辑，应该使用计算属性。

  • 计算属性默认值是getter，不过在需要时你也可以提供一个setter
