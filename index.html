Vue方法与事件处理

发表于 2017-02-05 | 分类于 Vue

方法与事件处理

方法处理器

- 可以用 v-on 指令监听 DOM 事件
- 在 methods 对象中定义方法

内联语句处理器

- 通过给事件直接传参

事件修饰器

- 在事件处理器中经常需要调用 event.preventDefault() 或 event.stopPropagation()，尽管可以通过处理 DOM 事件做到，但是，让方法是纯粹的数据逻辑会更好
- 为了解决这个问题，为 v-on 提供了两个事件修饰符
    1. .prevent: `@click.prevent="fn"`
    2. .stop: `@click.stop="fn"`
- 1.0.16 版本中添加了两个额外的修饰符
    1. 添加事件侦听器时使用capture: `@click.capture="fn"`
    2. 只对事件在元素本身(而不是子元素)触发时使用self: `@click.self="fn"`

按键修饰器

- 在监听键盘事件时，经常需要检测 keyCode，Vue 为 v-on 添加按键修饰符
- 通过检测相应 keyCode 数字
    

- 按键修饰器别名：enter、tab、delete、esc、space、up、down、left、right等
- 之后又新增了单字母按键别名、自定义按键别名等

为什么要在 HTML 中监听事件

- 轻松定位在 JavaScript 代码里对应的方法
- 无需在 JavaScript 里手动绑定事件，ViewModel 代码和 DOM 完全解耦
- 当一个 ViewModel 被销毁，所有的事件处理器都会自动被删除，无需担心如何手动清理
