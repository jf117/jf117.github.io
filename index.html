Vue数据绑定

发表于 2017-01-26 | 分类于 Vue

数据绑定语法

  • Vue.js 的模板是基于 DOM 实现的
  • 这就意味着所有的 Vue.js 模板都是可以解析的有效 html
  • Vue 模板因而从根本上不同于基于字符串的模板

插值文本

  • 数据绑定的最基础的形式是文本插值使用

  • 每次这个 data 的属性发生变化时它都会更新

  • 也可以只处理单次插值，之后的数据变化就不会引起插值

  • 双花括号可以将数据解析为纯文本，三花括号输出的是html字符串，会解析标签

  • 双花括号可用于html特性内，可用于动态修改标签的值

partial 模板片段

内容以 html 字符串插入，那么数据绑定将被忽略。如果需要复用模板片段，应当使用 partials。

  • 基本使用
  • 局部模板（动态模板）
  • 注意：
    1. 在网站上动态渲染任意 html 是非常危险的。
    2. 因为容易被攻击。
    3. 记住：只对可信内容使用html插值，永不用于用户提交的内容

绑定表达式

  • 放在双花括号标签内的文本称为绑定表达式。在Vue.js中，一段绑定表达式由一个简单的 JavaScript 表达式和可选的一个或多个过滤器构成。
  • 注意：
    1. 这些表达式将在所属的 Vue 实例的作用域计算。
    2. 一个限制是每个绑定只能包含单个表达式

二、You don’t have permission to access / on this server. 没权限

解决方案:

  1. 打开D:\wamp\bin\apache\Apache2.2.21\conf\httpd.conf
  2. 按ctrl+f 搜索documentRoot 大概188行左右的位置 directory标签添加 Allow from all

  4. 重启服务器
