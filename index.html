Vue表单控件与绑定

发表于 2017-02-06

表单控件与绑定

基础用法

  • 可以用 v-model 指令在表单控件元素上创建双向数据绑定。根据控件类型它自动选取正确的方法更新元素。
  • 一般与表单控件进行数据双向绑定，表单控件可使用的类型包括：
    • text： 双向绑定的值为输入表单的值
    • checkbox: 双向绑定的值为 value 属性的值
    • radio: 双向绑定的值为 value 属性的值
    • select: 双向绑定的值为 value 属性的值

绑定 value

  • 对于单选按钮，勾选框及选择框选项， v-model 绑定的 value 通常是静态字符串

  • 但是有时我们想绑定的 value 到 Vue 实例的一个动态属性上，这时可以 v-bind 实现，并且这个属性的值可以不是字符串

    • checkbox

    • radio

    • select option

参数特性

  • lazy

    • 在默认情况下， v-model 在 input 事件中同步输入框的值与数据，可以添加这个特性，从而改到在 change 事件中同步

  • number

    • 自动将用户的输入保持为数字

  • debounce

    • 设置最小延时，在每次敲击后延时同步输入框的值和数据。如果每次更新都要进行高耗操作(例如在输入提示中Ajax请求)
    • debounce 参数不会延迟 input 事件，它延迟”写入”底层数据。因此在使用 debounce 时应当用 vm.$watch() 响应数据的变化。若想延迟 DOM 事件，应当使用 debounce 过滤器。
